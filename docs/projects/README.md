## Как завести новый

1. Скопировать `_template.md` → `PXX-краткое-имя.md`
2. Привязать к цели из `docs/goals.md`
3. Добавить строку в таблицу выше
4. Разбить первую задачу на минизадачи — по одной сессии каждая
